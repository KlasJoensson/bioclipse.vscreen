About VScreen

Overview

Vscreen is a feature (= collection of plugins) for virtual screening of chemical databases. It facilitates setting up filters to drill down and isolate interesting compounds based on an extensible set of filters. By using the plugin-architecture of Bioclipse, it is straightforward to add new filters to VScreen.
